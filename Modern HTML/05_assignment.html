Semantic Tag

Semantic Tags are introduced in HTML 5 to use for specific content instead of div and span tag. Ex-header-tag,footer-tag,section-tag,nav-tag,main-tag etc.

Audio and Video Support

New version of audio and video tags are introduced for supporting different format of audio and video type(mp3,mp4,ogg,webm format) by the browser.

Responsive Image

Before to introduce of HTML5 version,it is necesaary to write more lines of CSS & JS code for creating responsive images.But now in HTML5 version,there is a new attribute introduced called "srcset" to specify multiple versions of an image at different screen resolutions.

Form enhancements

Due to Some new atrributes like placeholder,autocomplete,download,autofocus,required etc are introduced in HTML 5 ,it enhanced the functionality and behaviour of web elements.

Canvas Elements

Canvas Tags are used for graphics work related like Game development Purpose.
